# React Native SDK TypeScript example

### Installation

1. Install dependencies

   YARN:

   `$ yarn`

   NPM:

   `$ npm install`

2. Install PODS:

   `$ cd ios && pod install`

3. Enter secret and clientId in `src/credentials.json`

3. Run

   YARN:

   `$ yarn ios|android`

   NPM:

   `$ npm run ios|android`
